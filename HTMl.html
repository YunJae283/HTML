What is HTML?

HTML is the "HyperText Markup Language." We use this when we use documents. I hope you enjoy my HTML lesson!

Downloading the Editor

I use an editor called "atom." Download the editor on your computer. Next,open the atom. Then make a folder inside the atom. Finally, make a file inside the folder called "index.html."

Codes

In the code box, type <!DOCTYPE html>
<html>
<head> <title>title</title>
<meta charset="utf-8"
</head>
<body>
text
</body>
</html>.

In the <body> </body,>, write everything you want. If you want to make a text bold, write inside <strong>. Make sure you use / at the closing tag. If it's a paragraph, write inside <p> </p>. To make a hyperlink, write inside <a href="hyperlink"></a>.use <br> to switch lines.
